                         O QUE SIGNIFICA CADA COISA
================================================================================

CORES NOS MAPAS:
  VERDE = Hotspot REAL (fogo confirmado) - USE
  VERMELHO = Hotspot ESPURIO (falso positivo) - DESCARTE
  LARANJA = INCERTEZA (revisar manualmente) - REVISAR

GRAFICOS:
  Histogramas = Distribuicao de valores
  Scatter plots = Pontos no espaco
  Mapas = Localizacao geografica
  Graficos de linha = Mudancas ao longo do tempo
  Matriz de confusao = Acertos vs erros

SIMBOLOS:
  + = Coisa boa, fazer assim
  - = Coisa ruim, evitar
  ? = Cuidado, revisar manualmente
  # = Dados visualizados
  @ = Localizacao geografica

NUMEROS:
  0-1 = Probabilidade (0 = sem chance, 1 = certeza total)
  0-100 = Percentual ou confianca FIRMS
  HIGH/MEDIUM/LOW = Nivel de confianca na predicao
